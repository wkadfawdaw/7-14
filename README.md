# FundInsight（基金实时估值 Android App）

这是一个使用 **Kotlin + Jetpack Compose + MVVM + Retrofit** 的示例项目，覆盖你提到的核心需求：

1. 通过基金代码/名称搜索基金
2. 查看基金持仓
3. 查看历史净值
4. 其他功能（参考养基宝）：自选基金 + 涨跌提醒入口

## 功能说明

- **搜基金**：输入基金代码或关键字，点击搜索，支持点击卡片进入详情。
- **持仓&净值**：查看基金的持仓明细和历史净值变化。
- **自选&提醒**：展示自选基金列表与提醒开关（示例 UI）。

> 当前网络接口使用 `https://api.example.com/` 作为占位地址，仓库内提供了本地样例数据回退逻辑，便于直接跑通界面。

## 可扩展建议

- 对接真实基金接口（天天基金/聚合 API/券商开放平台）
- 引入 Room 做本地缓存和离线浏览
- 加入折线图（MPAndroidChart/Compose Charts）展示净值与估值
- 实现估值提醒（WorkManager + Notification）
- 增加组合收益分析、基金对比、回撤与夏普比率等高级指标

## 运行

1. 使用 Android Studio Hedgehog 或更高版本打开项目
2. 同步 Gradle
3. 运行 `app` 模块到模拟器或真机
